# 🏦 Bank Management System (Java)

## 📌 Project Overview
This is a desktop-based Bank Management System developed using Java. It simulates basic banking operations like account creation, login, deposit, withdrawal, and balance enquiry.

## 🚀 Features
- User Login & Signup
- Account Management
- Deposit & Withdrawal
- Balance Enquiry
- Mini Statement
- GUI using Swing

## 🛠️ Technologies Used
- Java (Core + Swing)
- MySQL (Database)
- JDBC (Database Connectivity)

## 📂 Project Structure
- `Login.java` – User authentication
- `Signup.java` – Account creation
- `Transactions.java` – Banking operations

## ⚙️ Setup Instructions
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Configure MySQL database
4. Add MySQL Connector JAR
5. Run `Main.java`

## 🗄️ Database Setup
- Create database: `bankmanagementsystem`
- Import SQL file (if available)
